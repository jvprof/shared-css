# Shared CSS Repository

Stylesheet et JavaScript partagés pour les projets éducatifs pages_web.

## Structure

- `css/style.css` - Stylesheet minimaliste académique (fun + IT)
- `js/script.js` - Validation sécurisée et gestion des modales vidéo

## Utilisation

### Développement local
```html
<link rel="stylesheet" href="../../shared/css/style.css">
<script src="../../shared/js/script.js"></script>
```

### Sur GitHub Pages
```html
<link rel="stylesheet" href="https://jvprof.github.io/shared-css/css/style.css">
<script src="https://jvprof.github.io/shared-css/js/script.js"></script>
```
